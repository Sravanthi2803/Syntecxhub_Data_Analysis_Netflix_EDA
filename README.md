# Netflix Data Analysis – Syntecxhub Internship

## Project Overview
This project is part of the Syntecxhub Data Science Internship. The goal of this project is to perform Exploratory Data Analysis (EDA) on the Netflix dataset and visualize important insights.

## Tools and Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Movies vs TV Shows comparison
- Content release trend over years
- Top 10 genres on Netflix
- Data visualization using graphs

## Key Insights
- Netflix has more Movies than TV Shows.
- Netflix content increased rapidly after 2015.
- Drama and International Movies are the most popular genres.
- Netflix content production has grown significantly in recent years.

## Conclusion
This project helped in understanding data analysis, data visualization, and extracting insights from real-world datasets using Python.
